Typography

Three families. DM Sans body, Manrope display (h1/h2 + pane heroes), Geist Mono for terminal + code.

Spacing

4-px base scale. Use these tokens only — never one-off values.

Radius

Four corner sizes. Most chrome lives between 6–14 px.

Elevation

Soft, paper-like. Never use harder Material-style shadows.

Motion

Tactile, not flashy. Use --ws-ease-out for most things; --ws-ease-spring only for modal entrance.

Layout dimensions

Iconography

Custom WorkDesk set — 24×24 viewBox, 1.75px stroke, rounded caps. Match this style when adding new glyphs.
