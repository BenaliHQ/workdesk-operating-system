The ribbon's primary nav element. One per zone; pastel dot with zone glyph. Active state shows a 3-px slate accent rail on the left edge and a soft outline around the dot.

.callout.{note|warning|success|info|error}

Callouts

Standalone blocks inside the editor body. Border-left rail in the type color, fill in the matching pastel zone-bg.
Note

A neutral observation. Uses the atlas blue.

Success

The thing worked. Uses gtd green.

Warning

Heads up. Uses intel ochre.

Info

Side-channel info. Uses config violet.

Error

Something is wrong. Uses personal rose.

Geometry

radius 8, border-left 3 px, padding 12 / 16
title
uppercase 12 px / 600, letter-spacing 0.06em, gap 6 px before icon
